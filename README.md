# xltool - 象限任务管理器（Android 1.0）

本仓库实现一个基于“四象限时间管理法”的 **MVP 效率工具**，优先交付 Android 版本 1.0（Kotlin + Jetpack Compose + Room）。

## 1. MVP 需求说明

详见：`docs/mvp.md`

## 2. Android 版本（1.0）能力范围

- **四象限主视图**：2x2 象限面板，显示象限标题与任务数量
- **任务管理**：新增 / 编辑 / 删除
- **任务属性**：标题、描述、预计耗时（分钟）、截止日期（可选）、标签（逗号分隔）、状态、象限
- **象限迁移**：通过任务菜单“移动到…”进行跨象限移动
- **排序与置顶（MVP）**：同象限内支持上移/下移；支持置顶（置顶任务始终排在前面）
- **本地存储**：Room（SQLite）

> 说明：PRD 中的“长按拖拽排序/跨象限拖拽”在后续版本可增强；1.0 先用“上移/下移 + 菜单迁移”确保稳定交付与低学习成本。

## 3. 运行方式（Android Studio）

1. 用 Android Studio 打开仓库根目录
2. 等待 Gradle Sync 完成
3. 选择 `app` 运行到模拟器或真机（最低支持 Android 8.0，minSdk=26）
